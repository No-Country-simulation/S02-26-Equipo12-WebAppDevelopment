# S02-26-Equipo12-WebAppDevelopment
RiderFit: Precision E-commerce for Equestrian Gear

FitBot es una herramienta inteligente que se integra con plataformas de e-commerce
para asistir a los compradores en la elección del talle correcto según sus medidas físicas.
En este caso, FitBot está específicamente orientada al mercado ecuestre, donde el proceso
de selección de productos requiere considerar no solo las medidas del jinete, sino también 
las características físicas del caballo. La herramienta permite que el usuario ingrese sus medidas
corporales junto con datos relevantes de su caballo, generando así un perfil combinado que representa 
la interacción real entre ambos durante la práctica ecuestre.
A partir de esta información, FitBot utiliza un asistente inteligente que cruza los datos 
ingresados con la base de productos disponible en cada e-commerce (tablas de talles, 
morfología del producto, materiales, cortes, disciplina de uso, entre otros). Como resultado,
el sistema recomienda los productos y talles que ofrecen el mejor ajuste posible para ese conjunto
jinete-caballo, reduciendo la incertidumbre en la compra online, disminuyendo devoluciones
y mejorando la experiencia del usuario. El objetivo de FitBot es transformar la compra de 
equipamiento ecuestre en una experiencia más precisa, personalizada y confiable, aportando
valor tanto a los consumidores como a las tiendas y marcas del sector.
